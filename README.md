# How to use:
Compile using `./gradlew build` then run `java -jar ./build/libs/MCS-1.0-SNAPSHOT.jar <DatapackPath>`

## Args
Possible arguments are
-o, --output: Specifies the Path where the Finished data folder will be stored
--obfuscate: Names every function just `function${id}`
--ast: Prints a Debug View of the AST into the Console
-v, --verbose: Makes the Output Verbose
